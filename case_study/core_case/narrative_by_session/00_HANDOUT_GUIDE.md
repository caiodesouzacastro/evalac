# Narrative Handout Guide — Creciendo Juntos

This folder contains 5 handouts, each extracting only the narrative sections
relevant to that session's exercises. They replace the full narrative document,
which is too long to read at once.

## Handout Summary

| File | Contains | When to Distribute | Format |
|------|----------|-------------------|--------|
| **S1_Narrative_Handout.docx** | Political context, program overview (all 4 components), family listening sessions (summary) | **Before Session 1** — participants read in advance | Print (2 pages) |
| **S2_Narrative_Handout.docx** | Implementation structure, staffing/timelines/budget, decision-makers' questions | **During Session 2, Exercise 2** — reference while specifying indicators | Print (2 pages) |
| **S3_Narrative_Handout.docx** | Family listening sessions (full quotes), municipal variation, implementation challenges | **During Session 3, Exercise 1** — qualitative investigation context | Print (2 pages) |
| **S4_Narrative_Handout.docx** | Monitoring data sources, program dosage structure, outcome evidence framework | **Before Session 4, Exercise 1** — data context for quantitative exercises | Print (2 pages) |
| **S5_Narrative_Handout.docx** | Decision-makers' questions, political stakes, scale-up context | **During Session 5, Exercise 5** — communication to decision-makers | Slides or print (2 pages) |

## Design Principles

- Each handout is **max 2 printed pages** — focused excerpts, not the full narrative
- Sections are mapped directly from the cheat sheet `NARRATIVE REFERENCE` markers
- Each handout ends with a forward pointer to the next session
- S1 provides the broadest context; S2-S5 are progressively narrower and more specialized
- Some content (e.g., family voices) appears in both S1 (summary) and S3 (full), since S3 exercises require deeper engagement with the same material

## Distribution Options

- **Printed handout**: Recommended for S1-S4 so participants can annotate and refer back during exercises
- **On-screen / slides**: Acceptable for S5 since quotes are short and used in plenary discussion
- **Pre-session reading**: S1 and S4 should be distributed before the session begins
- **During-exercise reading**: S2, S3, and S5 are distributed at the start of the relevant exercise
