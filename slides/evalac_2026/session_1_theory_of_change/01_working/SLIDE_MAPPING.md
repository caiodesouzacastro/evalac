## Key structural changes

1. **Case study integration** — Old deck used in-class reading (Exercise 1). New deck introduces CJ as a 1-paragraph vignette (Slide 5) and weaves it through every content slide.

2. **COM-B deepened** — Old Slide 20 introduced COM-B briefly. New structure adds an applied slide (18) and a full exercise (19) building COM-B pathways.

3. **Assumptions restructured** — Old deck had 7 slides on assumptions (25–31) using a breastfeeding example. New deck consolidates into 3 slides (22–24) using CJ, with a clearer taxonomy (reach / capacity / behavior / outcome).

4. **Exercises redesigned** — All 4 old exercises revised. Exercise 4 changed from small-group to facilitated discussion. New Exercise 5 added (optional) as bridge to Session 2.

5. **Debrief slides added** — Old deck had no debrief slides. New deck adds 3 (Slides 12, 20, 26) to consolidate learning after each exercise block.

6. **M&E bridge created** — Old deck ended abruptly. New Block 5 (Slides 30–34) explicitly connects ToC to evaluation questions, monitoring, and the Session 2 Results Matrix.

7. **Reference appendix** — Old Slides 40–41 (summary tables) moved to back. 5 new reference slides added (38–44).
